# ZCook Architecture

## System Flow

Client
↓
API
↓
Data Pipeline
↓
Shelby Storage
↓
Query Engine
↓
Application

## Components

### Client
User interface for discovering recipes.

### API
Handles requests, recipe submission, and queries.

### Data Pipeline
Processes recipe datasets and normalizes ingredients.

### Shelby Storage
Stores structured datasets and enables fast indexing.

### Query Engine
Supports advanced recipe search and filtering.
